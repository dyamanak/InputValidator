InputValidator

InputValidatorとは簡単な設定だけで入力値の妥当性チェックを行う Java 用、または JavaScript 用ライブラリです。

サンプル

  1. クライアント側（HTML）に入力値の妥当性チェックを追加する

  2. サーバ側(Java)とクライアント側(HTML)に入力値の妥当性チェックを追加する

チュートリアル

レイアウト設定

  1. フォーカスを設定する
  2. 読取専用を設定する
  3. 入力欄にヒントを表示する
  4. エラーの表示場所を指定する
  5. ヒントの表示場所を指定する
  1. エラーのスタイルを指定する
  2. ヒントのスタイルを指定する
  3. フォーカスセルのスタイルを指定する
  4. 入力エラーセルのスタイルを指定する
  1. 設定ミスを検出する
    1. 要素名の誤り
    2. チェックルールの誤り
    3. グループ要素名の誤り

チェック設定

  1. 入力必須チェック
  2. 文字数上限チェック
  3. 文字数範囲チェック
  4. E-Mailチェック
  5. 日付チェック
  6. 数値チェック
  7. 数値範囲チェック
  8. 全角文字チェック
  9. 半角文字チェック
  10. 正規表現チェック
  1. ラジオボタンに設定する
    1. 要素名が同じパターン
    2. 要素名が異なるパターン
  2. チェックボックスに設定する
    1. 要素名が同じパターン
    2. 要素名が異なるパターン
    3. 必須選択パターン
  3. リストボックスに設定する
    1. １つ選択するパターン
    2. 複数選択するパターン
    3. valueがないパターン
  1. カスタムバリデータを設定する
    1. 独自のチェックルールを追加する
    2. 独自のチェック関数を設定する
    3. パスワード２重チェック
    4. 複数フィールドチェック
    5. 動的にチェックを設定する

イベント設定

  1. ロード時のイベントを追加する
  2. 送信時のイベントを追加する
  3. リセット時のイベントを追加する
  4. チェック時のイベントを追加する
  1. 編集中のイベントを追加する
  2. 編集後のイベントを追加する
  1. 編集中にチェックする
  2. 送信時にまとめてチェックする

カスタムイベント一覧表(InputValidator.customEvent)

イベント 種類 記述例
onLoad Loadイベント 
onLoad : function() {
	alert('画面の初期処理があれば、ここに書きます。');
}
onSubmit Submitイベント 
onSubmit : function(evt, elements) {
	var errorCount = elements.length;
	if (errorCount > 0) {
		alert(errorCount + '件の入力エラーがあるため、登録できません。');
		return false;
	} else {
		if (confirm('入力内容を登録します。よろしいですか？')) {
			return true;
		} else {
			return false;
		}
	}
}
onReset Resetイベント 
onReset : function(evt) {
	if (confirm('入力内容を元に戻します。よろしいですか？')) {
		return true;
	} else {
		return false;
	}
}
onChange Changeイベント 
onChange : function(evt) {
	document.getElementById("editFlag1").innerHTML = "編集中";
},
onCheck Checkイベント 
onCheck : function(elements) {
	var errorCount = elements.length;
	if (errorCount > 0) {
		alert(errorCount + "件の入力エラーがあります。");
	} else {
		alert("入力エラーはありません。");
	}
}
